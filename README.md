# Visualizations with Matplotlib and Seaborn

This repository showcases data visualization examples using **Matplotlib** and **Seaborn** in Python.  
It demonstrates how to create clear, insightful, and aesthetically pleasing plots for exploratory data analysis and presentation.

---

## 📊 Overview
- Basic plotting with Matplotlib
- Advanced styling and customization
- Statistical visualizations with Seaborn
- Combining multiple libraries for richer insights

---

## 🛠️ Requirements
- Python 3.x
- Matplotlib
- Seaborn
- NumPy
- Pandas
